Akyral

web components for the modern world

Akyral isn't trying to be a framework or a library of any sorts. It isn't trying to replace the already well-established css and js frameworks out there.

Instead, Akyral aims to provide a set of commonly used web concepts with a ridiculous amount of customization.

Mobile First

Components are designed and constructed with a mobile first methodology.

Simple

Akyral focuses on simple to use semantic markup. Each component has its own declaritive attribute based api.

Performant

Each animation is chosen to perform above 60fps, ensuring a high quality feel.
