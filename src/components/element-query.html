[ Element Query ] What is it?

Where as a media query applies to the viewport of a device or page, an element query is affected by its individual dimensions on the page.

Helpful Gotchas

Parent Composition

{{component.package}} requires its parent css position be set to absolute or relative to work correctly.

Circularity

Be aware of circular style references in element query styles @see. One style may cause cycling in-between another and back again. Avoid defining a fixed width in components;
