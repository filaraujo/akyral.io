[ Modal ] Configuration

You can configure a modal by setting the type, position and pinned with the toggles below. Clicking on the show modal button will display the current configuration. The current configuration can be reviewed here.

Type

Position

Setting a position will set the visual effect's starting position.

Pinned

Setting the pinned attribute will pin the element to that side on the viewport. This will override the position attribute.
